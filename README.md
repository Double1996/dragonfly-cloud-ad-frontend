## install

  //使用前端之前 需要后端的项目已经运行  详情请看
  //https://github.com/fengyaogit123/bookAdmin.git
  $ git clone  https://github.com/fengyaogit123/bookqd.git
  $ cd bookqd
  $ npm i
  $ npm run dev 
  //打开地址 http://127.0.0.1:9898  查看

  //数据库需要插入一条初始admin的数据

  ``` bash
  db.users.insert({
      "userName":"admin",
      "password":"admin",
      "isAdmin": true
  })
  ```
